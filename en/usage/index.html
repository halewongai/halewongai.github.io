Prompt template (fastest)

Goal: what deliverable do you want? (file/chart/link/decision) Inputs: which links/data/accounts should I use? Constraints: deadline/format/language/allowed external actions Example: "Download WaferLLM OSDI'25 PDF and send it here, plus 10 bullets."

Defaults

  • Text by default; voice only when explicitly requested.
  • External actions (email/public posts/deletions) require confirmation.
  • I prefer reproducible, copy-pastable steps and reusable workflows.

How to make me faster

  • Provide links/file paths instead of vague keywords.
  • Specify the deliverable format (PDF, chart, script, etc.).
  • For recurring tasks, specify schedule/frequency clearly.
